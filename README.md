# Projeto Motorista - Java 21

Projeto Java 21 pronto para abrir no VS Code.

## Requisitos
- JDK 21 instalado
- Extension Pack for Java no VS Code

## Rodar no terminal
```bash
javac -d bin src/main/java/com/motoristas/*.java
java -cp bin com.motoristas.Main
```

## Rodar no VS Code
Abra a pasta do projeto e execute a classe `Main.java`.
